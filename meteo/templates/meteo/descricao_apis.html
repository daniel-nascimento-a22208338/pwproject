Endpoints da API Interna

1. Listagem de Cidades

Descrição: Este endpoint retorna uma lista de todas as cidades disponíveis na API.

URL: /api/cidades

Método: GET

Exemplo de Resposta:

{
  "cidades": [
    "Lisboa",
    "Porto",
    "Madrid",
    "Paris",
    "Nova York"
  ]
}

2. Previsão Meteorológica de Hoje para uma Cidade

Descrição: Este endpoint retorna a previsão meteorológica para hoje de uma cidade específica.

URL: /api/previsao/hoje

Método: GET

Parâmetros de Query: cidade (string): Nome da cidade para a qual a previsão é solicitada.

Exemplo de Resposta:

{
  "cidade": "Lisboa",
  "data": "2024-06-05",
  "temperatura_minima": 18.5,
  "temperatura_maxima": 25.3,
  "descricao_tempo": "Parcialmente nublado",
  "precipitacao": 0,
  "icone_url": "https://example.com/icons/partly_cloudy.png"
}

3. Previsão Meteorológica para os Próximos 5 Dias para uma Cidade

Descrição: Este endpoint retorna a previsão meteorológica para os próximos 5 dias de uma cidade específica.

URL: /api/previsao/proximos5dias

Método: GET

Parâmetros de Query: cidade (string): Nome da cidade para a qual a previsão é solicitada.

Exemplo de Resposta:

{
  "cidade": "Lisboa",
  "previsoes": [
    {
      "data": "2024-06-05",
      "temperatura_minima": 18.5,
      "temperatura_maxima": 25.3,
      "descricao_tempo": "Parcialmente nublado",
      "precipitacao": 0,
      "icone_url": "https://example.com/icons/partly_cloudy.png"
    }
  ]
}
